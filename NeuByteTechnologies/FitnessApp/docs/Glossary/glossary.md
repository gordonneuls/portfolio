# Glossary of Terms

| Term                             | Definition                                                                                        |
|----------------------------------|---------------------------------------------------------------------------------------------------|
| RPE (Rate of Perceived Exertion) | A subjective 1–10 scale measuring how difficult a set feels. 1 = very light; 10 = maximal effort. |
| 1RM (One‑Rep Max)                | The maximum amount of weight a user can lift for one repetition of an exercise.                   |
| Set                              | A group of consecutive repetitions of an exercise.                                                |
| Rep (Repetition)                 | One complete movement of an exercise.                                                             |
| Superset                         | Two or more exercises performed back‑to‑back with minimal rest.                                   |
| Circuit                          | A sequence of exercises performed in rotation with limited rest.                                  |
| Program                          | A structured multi‑week training plan consisting of workouts, days, and exercises.                |
| Workout                          | A single training session containing one or more exercises.                                       |
| Exercise Summary                 | A short description of an exercise including cues, equipment, and instructions.                   |
| PR (Personal Record)             | The highest weight, reps, or performance achieved by a user for a given exercise.                 |
| Previous Performance             | View previous logged exercise metrics to inform progression over time                             |
